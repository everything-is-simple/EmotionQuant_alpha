### 2.2 订单处理详细流程

```
收到 TradeSignal
     │
     ▼
┌─────────────────────────────────────────┐
│  Step 1: 信号验证 (v2.0)                │
│  validation = validator.validate_signal(│
│      signal, mss, irs, pas              │
│  )                                       │
│                                          │
│  if not validation.is_tradeable:        │
│      return Rejected("质量不足")         │
└─────────────────┬───────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│  Step 2: 计算调整后仓位                  │
│  adjusted_size = signal.position_size   │
│                  × validation.position_adjustment│
│                                          │
│  shares = calculate_shares(             │
│      cash, adjusted_size, signal.entry  │
│  )                                       │
└─────────────────┬───────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│  Step 3: 风控检查                        │
│  order = Order(shares, entry, ...)      │
│  passed, reason = risk.check_order(     │
│      order, positions, cash, equity     │
│  )                                       │
│                                          │
│  if not passed:                         │
│      return Rejected(reason)            │
└─────────────────┬───────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│  Step 4: 提交订单                        │
│  order_manager.submit_order(order)      │
│  order.status = "submitted"             │
└─────────────────┬───────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│  Step 5: 执行交易（集合竞价）             │
│  executed = executor.execute_auction(   │
│      order, trade_date                  │
│  )                                       │
│                                          │
│  filled_price = open_price               │
│  slippage = 0                            │
│  commission = calculate_fee(amount)     │
└─────────────────┬───────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│  Step 6: 更新状态                        │
│                                          │
│  if direction == "buy":                 │
│      position_manager.add_position(     │
│          stock_code, shares, price,     │
│          stop_price, target_price       │
│      )                                   │
│      t1_tracker.buy(stock_code, shares) │
│      cash -= (amount + fee)             │
│  else:                                  │
│      position_manager.reduce_position() │
│      cash += (amount - fee)             │
└─────────────────┬───────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│  Step 7: 持久化                          │
│  save_trade_record(executed)            │
│  save_positions()                       │
│  return executed                         │
└─────────────────────────────────────────┘
```

补充运行时输入（L1）：
- `raw_daily`：集合竞价成交价来源（开盘价）
- `raw_limit_list`：涨跌停状态检查（TuShare limit_list_d 落库）

信号桥接规则（L3 Integration → Trading）：
- `integrated_recommendation.direction`（bullish/bearish/neutral）映射到 `TradeSignal.direction`（buy/sell）；neutral 直接过滤不下单。
- `integrated_recommendation.integration_mode` 原样透传至 `TradeSignal.integration_mode`，通过 `signal_id` 可回溯来源。
- `entry/stop/target` 优先使用集成输出；缺失时由交易风控参数补齐。

| trading_state | 触发条件 | 行为 |

## 6. 异常处理

### 6.1 数据缺失处理

| 缺失数据 | 处理策略 |
|----------|----------|
| Validation Gate = FAIL | 跳过当日信号生成，记录 `trading_state=blocked_gate_fail` |
| mss_panorama | 使用前一日数据，记录 `trading_state=warn_data_fallback` |
| irs_industry_daily | 允许所有行业 |
| integrated_recommendation | 当日不产生信号 |
| 开盘价 | 使用前收盘价 |

### 6.2 执行异常处理

| 异常情况 | 处理策略 |
|----------|----------|
| 资金不足 | 拒绝订单，`reject_reason=REJECT_NO_CASH` |
| 超单股仓位上限 | 拒绝订单，`reject_reason=REJECT_MAX_POSITION` |
| 超行业仓位上限 | 拒绝订单，`reject_reason=REJECT_MAX_INDUSTRY` |
| 超总仓位上限 | 拒绝订单，`reject_reason=REJECT_MAX_TOTAL_POSITION` |
| T+1限制 | 拒绝订单，`reject_reason=REJECT_T1_FROZEN` |
| 涨停买入限制 | 拒绝订单，`reject_reason=REJECT_LIMIT_UP` |
| 跌停卖出限制 | 拒绝订单，`reject_reason=REJECT_LIMIT_DOWN` |
| 低可成交概率 | 拒绝订单，`reject_reason=REJECT_LOW_FILL_PROB` |
| 执行超时 | 重试3次后取消，并标记 `trading_state=blocked_untradable` |
